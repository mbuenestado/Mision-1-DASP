# Mision-1-DASP
Los prompts utilizados para la realización de la misión.

Actúa como un Desarrollador Web Senior y Diseñador de UX/UI experto en marcas de lujo. Mi objetivo es crear una Landing Page profesional para "Guau Couture", una firma de alta costura canina.

Directrices de Identidad Visual (ESTRICTO): * Colores: Principal `#8FBC8F` (Verde Menta), Acento `#E8B4B8` (Rosa Jardín), Fondo `#FAFAF8` (Blanco roto), Detalles en `#000000` (Negro Tinta). * Tipografía: `Playfair Display` para titulares (serif elegante) y `DM Sans` para cuerpo de texto y navegación.

Voz y Tono: Formal, aristocrático, con ironía sutil. Como un cronista del siglo XIX. Evita exclamaciones. Usa términos como "refinados", "de alcurnia" o "impecable porte". * Tagline: "La distinción no se compra. Se hereda. O se viste".

Estructura de la Landing (Artefacto de Claude):

•	Hero Section: Imagen sugerida de alta calidad, titular elegante y botón de "Reservar Asiento".
•	Descripción del Evento: Texto que narre el desfile del Viernes Santo en la Plaza San Francisco de Sevilla.
•	Programa del Desfile (4 Pases): > * I: Armados de la Macarena (Galgos). * II: Gran Power (Mastines). * III: Trianeros (Perros de Agua). * IV: Los Cachorros (Bodegueros y Bretones).
•	Información Práctica: 3 de abril de 2026, 9:00 a 11:00h, Plaza San Francisco. 5. Footer: Registro con formulario minimalista.

Por favor, genera el código HTML/Tailwind CSS en un artefacto de Claude que sea visualmente impresionante, minimalista y cumpla con el manual de estilo.

Prompt de Refinamiento: El resultado es bueno, pero vamos a elevar la sofisticación para que parezca una web de "pago":

•	Estética: Añade bordes finos (1px) en color negro, mucho espacio en blanco (padding) y asegúrate de que el contraste entre el verde menta y el rosa sea sutil.
•	Interactividad: Haz que las tarjetas de los pases del programa tengan un efecto de "hover" suave.
•	Copy: Asegúrate de que no haya ni una sola exclamación. Cambia cualquier lenguaje común por términos más elevados (ej: en lugar de "ven a ver el desfile", usa "le invitamos a presenciar la liturgia de la moda canina").
•	Imágenes: Añade placeholders de imágenes con el tamaño adecuado (`/api/placeholder/800/400`) para que la estructura sea perfecta.
